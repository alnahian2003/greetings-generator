## Installation

You can install the package via composer:

```bash
composer require alnahian2003/greetings-generator
```

## Usage

```php
use Alnahian2003\GreetingsGenerator\Greetings;

echo Greetings::greet();
```

## Testing

```bash
composer test
```

or you can also try this

```bash
./vendor/bin/phpunit
```
